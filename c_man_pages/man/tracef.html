NAME

tracef − LTTng−UST printf(3)−like interface

SYNOPSIS

#include <lttng/tracef.h>

#define tracef(fmt, ...)

Link with -llttng-ust -llttng-ust-common.

DESCRIPTION

The tracef() macro is part of version 0 of the LTTng−UST API (see the “Compatibility with previous APIs” section of lttng-ust(3)).

Prefer using lttng_ust_tracef(3) which is part of version 1 of the API and has the lttng_ust_ namespace.
