NAME

grub-get-kernel-settings − Evaluate the system’s kernel installation settings for use while making a grub configuration file

SYNOPSIS

grub-get-kernel-settings [OPTION]

DESCRIPTION

Evaluate new−kernel−pkg configuration
−h, −−help

print this message and exit

−v, −−version

print the version information and exit

SEE ALSO

The full documentation for grub-get-kernel-settings is maintained as a Texinfo manual. If the info and grub-get-kernel-settings programs are properly installed at your site, the command

info grub-get-kernel-settings

should give you access to the complete manual.
